Users configuration

To allow a user to connect to the VPN , you must add it to the group "VPNusers" in the tab Users of MDS.


To withdraw access, simply exclude it from the group.
current connections are not interrupted, this change will apply only to future connections.

Server configuration

Everything is done thanks to MSS, the parameters can be entered in the MMI.

Client configuration

For Windows clients, you need to download the client at http://openvpn.net/index.php/download/community-downloads.html

the installer is little bit buggy, it seems it does not clean the registry, so if it must be reinstalled the new installation may use the registry keys from the previous which is not what we want.

Run the setup in administrator and validate the default choices to complete the installation.

A configuration file for the client is automatically generated.
It named mbs_vpn_client.ovpn and is located in a archive that can be downloaded from MSS .



The files mbs_vpn_client.ovpn and ca.crt, contained in the archive, should be decompressed in the client configuration folder, it is accessible from the Windows menu.


The application "openvpn_gui" must be launched as Administrator (the first time, in Windows 7 (right click on the item then "run as Administrator"))

Select the icon in the taskbar.

Right click then "connect". The name and password of an authorized user has to be filled.


The connection is made , the client can access resources present on the VPN . The OpenVPN icon turns green :
