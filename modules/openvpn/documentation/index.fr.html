Configuration des utilisateurs

Pour autoriser un utilisateur à se connecter au VPN, il faut l'ajouter au groupe VPNUsers dans l'onglet Users de MDS.


Pour lui retirer l’accès il suffit de l'exclure du groupe.
les connexions en cours ne sont pas interrompues, cette modification s'appliquera au connexions futures.

Configuration serveur

Tout est fait grace à MSS, les paramètres peuvent etre saisie dans l'ihm.

Configuration poste client

Pour les clients windows, il faut télécharger le client qui se trouve http://openvpn.net/index.php/download/community-downloads.html

l'installeur est sensiblement bugué, il semble qu'il ne sache pas bien nettoyer la base de registre ainsi s'il doit etre réinstallé la nouvelle installation risque d'utiliser les clés de registre de la précédente.

Lancer le setup en Administrateur valider les choix par défaut pour finaliser l'installation.

Un fichier de configuration pour les clients est automatiquement généré.
Il se nomme mbs_vpn_client.ovpn et se trouve dans une archive téléchargeable depuis MSS.



Les fichiers mbs_vpn_client.ovpn et ca.crt présents dans l'archive de configuration doivent etre décompressées dans le dossier de configuration du client accessible depuis le menu de Windows.


l'application openvpn_gui doit etre lancée (la première fois en Administrateur sous Windows 7 (click doit lancer en tant qu'Administrateur))

Sélectionner l'ihm dans la barre des tâches.

Click droit puis "connnecter" Le nom et le mot de passe d'un utilisateur autorisé à se connecter au VPN sont alors demandés et doivent etre saisis.
